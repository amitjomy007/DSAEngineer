## 🏗️ Technical Architecture

**Stack**  
- **Frontend:** React (TypeScript)  
- **Backend:** Express.js  
- **Deployment:**  
  - Frontend: **Vercel**  
  - Backend: **Railway**  
- **Compiler Infrastructure:**  
  - Dockerized compiler hosted on **Amazon EC2**  
  - Images stored in **Amazon ECR**
